##steps for installation:

- npm i
- create env file with below details:
BASE_URL=http://20.244.56.144/test
PORT=3000
CACHE_TTL=300  # 5 minutes in seconds
AUTH_TOKEN=<auth_token>

- npm run dev
- run localhost at port 3000
